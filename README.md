# Luxio 

Dashboard web oscuro para registrar y visualizar gastos personales por categoría.

## Funciones

- Sidebar de navegación con secciones Principal y Cuenta
- Agregar gastos con descripción, monto y categoría
- Stats en tiempo real: total gastado, cantidad y mayor gasto
- Gráfica de barras por categoría
- Lista de últimos gastos con animación y opción de eliminar
- Pills de categorías activas y distribución porcentual
- Los datos se guardan automáticamente en el navegador (localStorage)

## Tecnologías

- HTML5
- CSS3 (dark theme, animaciones, responsive)
- JavaScript (lógica, localStorage)
- Tabler Icons
